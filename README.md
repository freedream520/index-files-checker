# index-files-checker
读取索引文件和实际文件存放文件夹，检查索引文件与实际数据文件的一致性，前提是需要满足索引文件中的文件名与实际文件文件名一致的条件。可输出至txt文件，并可指定路径。

在Python 2.7.13环境中测试可直接使用
